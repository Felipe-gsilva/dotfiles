# dotfiles

this is properly working only for arch linux

## installing 

just give the install.sh permissions 
```bash

$ chmod +x install.sh

$ ./install.sh

```

> [!info] caps2esc
> I'm using that `install.sh` alternative with setxkbmap -option "caps:swapescape" in my desktop, but I suggest you use interception-caps2esc
> which is much safer. Im using it at my notebook, but did not configure it here.
